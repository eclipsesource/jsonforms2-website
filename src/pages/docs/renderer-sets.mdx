# Renderer sets

This is an overview of all available renderer sets and which features they support.

<RendererSetSchemaFeatureTable />
<br />
<RendererSetUISchemaFeatureTable />

## React Material UI Renderer Set

The React Material UI Renderer Set is based on [Material UI](https://material-ui.com).

## React Vanilla Renderer Set

The React Vanilla Renderer Set is based on plain HTML with a set of custom styles.
For more information and a customization guide, see the [Vanilla Readme](https://github.com/eclipsesource/jsonforms/blob/master/packages/vanilla/README.md) and [Styles](https://github.com/eclipsesource/jsonforms/blob/master/packages/vanilla/Styles.md) manual.

## Angular Material UI Renderer Set

The Angular Material UI Renderer Set is based on [Angular Material](https://material.angular.io/).
