#  Getting Started

This section describes how you can get quickly started with JSON Forms.

1. Clone the [seed](https://github.com/eclipsesource/jsonforms-react-seed)  app with `git clone`
2. Install dependencies with: 
   ```bash
   npm install
   ```
3. Run the app with:
   ```
   npm start
   ```

For more info about the seed app, please see the corresponding README file of the
the [seed repo](https://github.com/eclipsesource/jsonforms-react-seed).

For a more detailed tutorial about the usage of JSON Forms, please
see [our tutorial](/docs/tutorial).
